## Laws

```
map-law1 : {A : Type} (xs : List A) → map (idfun A) xs ≡ xs
map-law1 [] = refl
map-law1 (x ∷ xs) i = x ∷ (map-law1 xs i)

map-law2 : {A B C : Type} (g : B → C) (f : A → B) (xs : List A)
         → map (g ∘ f) xs ≡ map g (map f xs)
map-law2 g f [] = refl
map-law2 g f (x ∷ xs) i = g (f x) ∷ (map-law2 g f xs i)
```
